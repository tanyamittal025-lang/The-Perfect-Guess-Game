# The-Perfect-Guess-Game
The Perfect Guess Game is a simple number guessing game built using Python. In this game, the computer randomly selects a number between 1 and 100, and the player has to guess the correct number.  The program provides hints after each guess to help the player reach the correct answer.

##  How the Game Works

1. The computer randomly generates a number between **1 and 100**.
2. The player enters a guess.
3. The program gives feedback:

   * **"Higher number please"** if the guess is too low.
   * **"Lower number please"** if the guess is too high.
4. The game continues until the player guesses the correct number.
5. At the end, the program shows the **number of guesses taken**.
